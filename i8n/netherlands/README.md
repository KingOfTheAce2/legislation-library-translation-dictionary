# Netherlands Legal Dictionary

This directory contains legal dictionaries and legislation specific to **Netherlands law and jurisdiction**.

## Current Files

### Monolingual Dutch Dictionaries

- **NL-legal-dictionary.json** (11 KB)
  - 495+ Dutch legal terms with Dutch definitions
  - Source: Raad van de Rechtspraak (Dutch Judiciary Council)
  - License: CC0 (Public Domain)

## Jurisdiction Scope

**IMPORTANT:** This content is specifically for **Netherlands law as applied in the Netherlands**.

It does NOT cover:
- Belgian law (even though Belgian Dutch is similar)
- Suriname law
- Other Dutch-speaking regions

All terminology, definitions, and translations are specific to the Dutch legal system.

## File Naming Convention

All files follow the pattern: `[SOURCE-LANG]-[TARGET-LANG]-[CONTENT-TYPE].json`

Examples:
- `NL-EN-legal-dictionary.json` = Dutch → English legal dictionary
- `NL-legal-dictionary.json` = Dutch monolingual (NL-NL)
- `NL-FR-legal-glossary.json` = Dutch → French glossary

## Usage in Website

These files are loaded by:
- `index.html` - Main dictionary interface (uses unified-dictionary.json which aggregates these)
- `legislation.html` - Civil Procedure browser (uses NL-EN-civil-procedure-all.json)

## Processing Scripts

Located in `/import/` directory:
- `parse_tmx.py` - Import TMX translation memories
- `extract_docx_and_fix.py` - Extract from DOCX files
- `process_legislation.py` - Process legislation data
- Various book-specific parsers
